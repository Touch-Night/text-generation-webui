# 键盘快捷键

#### 通用

| 快捷键                   | 描述                                             |
|-------------------------|--------------------------------------------------|
| Esc                     | 停止生成                                          |
| Tab                     | 在当前标签页和参数标签页之间切换                   |

#### 聊天标签页

| 快捷键                   | 描述                                             |
|-------------------------|--------------------------------------------------|
| Ctrl + S                | 展示/隐藏聊天控件                                 |
| Ctrl + Enter            | 重新生成                                          |
| Alt + Enter             | 继续                                             |
| Ctrl + Shift + Backspace| 删除上一条回复                                    |
| Ctrl + Shift + K        | 复制上一条回复                                    |
| Ctrl + Shift + L        | 替换上一条回复                                    |
| Ctrl + Shift + M        | AI帮答                                           |
